# Jarvis Bot 🎤💻

A voice-controlled screenshot & screen recorder built with Python.

## Features
- Take screenshots by saying: **"Hey Jarvis, take screenshot"**
- Start screen recording: **"Hey Jarvis, start recording"**
- Stop recording: **"Hey Jarvis, stop recording"**
- Exit: **"Hey Jarvis, exit"**

## Setup
```bash
git clone <repo>
cd jarvis_bot
pip install -r requirements.txt
python main.py


💡 Jarvis commands:

“Take screenshot” → Screenshot save
“Start recording” → Begin screen recording
“Stop recording” → Stop screen recording
“Hello” / “Hi” → Jarvis greeting
“Exit / Quit / Bye” → Exit program
